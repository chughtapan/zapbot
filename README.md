# Zapbot

Plan-to-code workflow for teams. Create plans in Claude Code, publish them as GitHub issues for team review via plannotator, and approved plans are automatically implemented by AI agents.

## Install

```bash
git clone https://github.com/chughtapan/zapbot ~/.claude/skills/zapbot
cd ~/.claude/skills/zapbot && ./setup
```

## Onboard a repo

```bash
cd ~/your-project
~/.claude/skills/zapbot/bin/zapbot-team-init
```

Creates `.agent-rules.md`, `agent-orchestrator.yaml`, `.env` (with random webhook secret), GitHub labels, and CLAUDE.md routing rules.

## Start

```bash
cd ~/your-project
~/.claude/skills/zapbot/start.sh
```

Starts the webhook bridge (port 3000), agent-orchestrator (port 3001), and ngrok tunnel.

For GCP/static IP (no ngrok): `~/.claude/skills/zapbot/start.sh --no-ngrok`

## Use it

In Claude Code:
```
/plan                    # Create a plan
/zapbot-publish          # Publish to GitHub issue with plannotator link
```

Team reviews the plan via plannotator. When ready, add the `plan-approved` label. An agent spawns in an isolated worktree and creates a PR.

## Upgrade

```bash
cd ~/.claude/skills/zapbot && git pull && ./setup
```

The skill checks for updates automatically.

## Configuration

Set in `.env` in your project repo (generated by `zapbot-team-init`):

| Variable | Default | Description |
|----------|---------|-------------|
| `GITHUB_WEBHOOK_SECRET` | (generated) | HMAC secret for webhook verification |
| `ZAPBOT_REPO` | auto-detected | GitHub repo (owner/name) |
| `ZAPBOT_BRIDGE_PORT` | 3000 | Webhook bridge port |
| `ZAPBOT_AO_PORT` | 3001 | Agent-orchestrator port |
| `ZAPBOT_APPROVE_LABEL` | plan-approved | Label that triggers implementation |
| `ZAPBOT_BRIDGE_URL` | (set by start.sh) | Public URL for plannotator callbacks |

## How it works

```
Developer (Claude Code)          GitHub               Server
  |                                |                      |
  | /zapbot-publish ──────────────>| Issue created         |
  |   + plan body                  | + plannotator link    |
  |   + share link                 | + zapbot-plan label   |
  |                                |                      |
  |                                | Team reviews          |
  |                                | via plannotator       |
  |                                |                      |
  |                                | plan-approved ────────>| webhook-bridge
  |                                | label added           | |-- ao spawn
  |                                |                      | |   (worktree)
  |                                |<── PR created ────────| '-- creates PR
  |                                |                      |
  | pull, review, merge            |                      |
```
